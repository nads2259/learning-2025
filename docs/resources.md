# Resource Guide

## Core Texts & References
- **Networking:** "Computer Networking: A Top-Down Approach" (Kurose & Ross)
- **Systems & OS:** "Operating System Concepts" (Silberschatz et al.)
- **Software Engineering:** "Clean Code" (Robert C. Martin), "Accelerate" (Forsgren et al.)
- **DevOps:** "The Phoenix Project" and "The DevOps Handbook" (Kim et al.)
- **Python:** "Fluent Python" (Luciano Ramalho), "Effective Python" (Brett Slatkin)
- **Data Analysis:** "Python for Data Analysis" (Wes McKinney), "Storytelling with Data" (Cole Nussbaumer Knaflic)

## Recommended Tools
- Shell: Bash, Zsh, PowerShell
- Editors/IDEs: VS Code, PyCharm, JupyterLab
- Version Control: Git, GitHub Desktop
- CI/CD: GitHub Actions, GitLab CI, CircleCI
- Cloud Providers: AWS Educate, Azure for Students, Google Cloud Skills Boost
- Data Tooling: pandas, NumPy, scikit-learn, Tableau Public, Power BI Desktop

## Practice Platforms
- Networking labs via Cisco Packet Tracer and TryHackMe
- Coding challenges on LeetCode, HackerRank, CodeSignal
- DevOps sandboxes on Katacoda, Play with Docker, KubeAcademy
- Data projects through Kaggle, data.gov, and Google Dataset Search
