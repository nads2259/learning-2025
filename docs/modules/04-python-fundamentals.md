# Module 4: Python Fundamentals

## Week 1: Core Syntax & Control Flow
- Install Python, set up virtual environments, manage dependencies
- Explore variables, data types, expressions, and operators
- Implement control flow with conditionals, loops, and comprehensions
- Lab: Build CLI utilities that process user input and files

## Week 2: Functions & Modular Programming
- Define functions, scope rules, and default/keyword arguments
- Use modules, packages, and import mechanisms effectively
- Document code with docstrings and type hints (PEP 257/484)
- Lab: Create a modular Python package with unit tests

## Week 3: Data Structures & File I/O
- Work with strings, lists, tuples, sets, dicts, stacks, queues
- Manage JSON/CSV data, file encodings, and context managers
- Handle exceptions gracefully and design custom error classes
- Lab: Develop a log parsing tool with error resilience

## Week 4: Testing & Debugging
- Apply pytest, unittest, and doctest to validate behavior
- Debug using logging, breakpoints, and profiling tools
- Integrate linting/formatting (flake8, black, isort, mypy)
- Lab: Build a mini ETL pipeline with automated tests and linters

## Assessments & Deliverables
- Weekly quizzes and coding challenges
- Python fundamentals practical exam
- Module portfolio: tested CLI application + documentation
