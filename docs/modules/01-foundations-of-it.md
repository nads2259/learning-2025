# Module 1: Foundations of IT & Networking

## Week 1: Digital & Operating System Fundamentals
- Understand binary, hexadecimal, and data representation
- Compare major operating systems and their architecture
- Practice terminal navigation, file permissions, and process management
- Lab: Set up a Linux environment, configure users, and schedule cron tasks

## Week 2: Networking Building Blocks
- Explain IP addressing, subnetting, and CIDR notation
- Analyze client-server models and communication patterns
- Map data flow through the TCP/IP stack and OSI layers
- Lab: Use Wireshark/cURL/netcat to inspect network traffic

## Week 3: Core Services & Protocols
- Explore DNS, DHCP, HTTP/S, FTP, SMTP, and SSH
- Configure local services and interpret protocol headers
- Study routing, switching, and load balancing fundamentals
- Lab: Build a virtual network, implement routing rules, and test latency

## Week 4: Security Essentials
- Identify threat models, vulnerability classes, and hardening techniques
- Configure firewalls, VPN tunnels, and multi-factor authentication
- Understand encryption (symmetric/asymmetric), certificates, and TLS
- Lab: Perform a basic security audit and document mitigation plans

## Assessments & Deliverables
- Weekly quizzes and labs
- Networking concept presentation
- Module portfolio: system architecture diagram + security hardening checklist
