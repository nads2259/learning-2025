# Module 6: Applied Data Analysis

## Week 1: Analytical Thinking & Data Acquisition
- Frame analytical questions and define success metrics
- Collect datasets from files, APIs, web scraping, and databases
- Manage data governance, ethics, and compliance
- Lab: Build a reproducible data collection notebook

## Week 2: Data Cleaning & Feature Engineering
- Handle missing values, outliers, and categorical encoding
- Apply normalization, scaling, and feature selection techniques
- Use pandas pipelines and custom transformers for repeatability
- Lab: Clean a messy dataset and document transformation logic

## Week 3: Exploratory Data Analysis & Visualization
- Summarize data using descriptive statistics and probability
- Visualize distributions, relationships, and time-series trends
- Employ matplotlib, seaborn, plotly, and dashboard frameworks
- Lab: Produce a storytelling dashboard with stakeholder annotations

## Week 4: Insights, Communication & Automation
- Build automated reporting pipelines and scheduled jobs
- Communicate findings through presentations and executive summaries
- Integrate BI tools (Power BI/Tableau/Looker) and notebooks
- Lab: Deliver a stakeholder presentation with actionable insights

## Capstone Integration
- Synthesize skills from previous modules into a polished analytics project
- Implement versioned notebooks, reproducible environments, and QA checks
- Present findings to a review panel with data-backed recommendations

## Assessments & Deliverables
- Weekly notebook submissions and peer reviews
- Dashboard demo and stakeholder Q&A
- Module portfolio: reproducible analytics project repository
