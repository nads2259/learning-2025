# Module 3: DevOps & Cloud Foundations

## Week 1: DevOps Culture & CI/CD
- Define DevOps principles and value streams
- Map software delivery pipelines and feedback loops
- Build CI workflows with automated testing and linting
- Lab: Configure a simple CI pipeline using GitHub Actions

## Week 2: Infrastructure as Code
- Understand configuration management tools (Ansible, Chef, Puppet)
- Author IaC templates with Terraform or CloudFormation basics
- Manage secrets, environment variables, and configuration drift
- Lab: Provision cloud infrastructure for a sample web application

## Week 3: Containers & Orchestration
- Build, tag, and publish Docker images following best practices
- Compare orchestration platforms (Kubernetes, ECS, Nomad)
- Implement service discovery, scaling policies, and health checks
- Lab: Deploy a containerized service and observe metrics/logs

## Week 4: Monitoring & Reliability
- Collect logs and metrics with ELK/EFK stacks and Prometheus/Grafana
- Apply SRE concepts: SLIs, SLOs, SLAs, error budgets
- Plan incident response, on-call rotations, and runbooks
- Lab: Create dashboards, alerts, and post-incident reviews

## Assessments & Deliverables
- CI/CD pipeline demonstration
- IaC infrastructure review
- Module portfolio: monitoring dashboard + incident management report
